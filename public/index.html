Guide d'Importation sur la Montre

PC / Mac (Web)

  1. Connectez-vous sur Garmin Connect Web.
  2. Dans la barre latérale, allez sur Entraînements & Planification → Parcours.
  3. Cliquez sur Importer (petit lien en bas à droite) et téléversez le fichier GPX ou TCX mis à jour.
  4. Sélectionnez le type de parcours : Trail running.
  5. Cliquez sur Enregistrer le nouveau parcours, puis synchronisez avec votre montre depuis l'application mobile Garmin Connect.

Android / iOS (Mobile)

  1. Envoyez le fichier GPX ou TCX sur votre smartphone (AirDrop, e-mail, messagerie...).
  2. Ouvrez le fichier sur votre téléphone et sélectionnez l'application Garmin Connect pour l'ouvrir.
  3. Choisissez le type de parcours : Trail running.
  4. Enregistrez le parcours. Appuyez sur les trois points (ou l'icône de montre) en haut à droite pour l'envoyer à votre montre, puis lancez une synchronisation.

PC / Mac (Web)

Suunto ne propose plus l'importation de parcours par site web. Veuillez utiliser l'application mobile Suunto.

Android / iOS (Mobile)

  1. Téléchargez ou envoyez le fichier GPX (Suunto GPX) sur votre smartphone.
  2. Ouvrez l'application Suunto sur votre téléphone.
  3. Allez dans l'onglet Carte, appuyez sur le bouton + et sélectionnez Importer GPX.
  4. Sélectionnez le fichier téléchargé, donnez un nom à l'itinéraire et activez l'option Utiliser dans la montre, puis synchronisez.

PC / Mac (Web)

  1. Connectez-vous sur le site Coros Training Hub.
  2. Allez dans l'onglet Bibliothèque de parcours et cliquez sur Importer le parcours.
  3. Téléversez le fichier GPX. Le parcours se synchronisera ensuite automatiquement avec votre application smartphone.

Android / iOS (Mobile)

  1. Téléchargez ou envoyez le fichier GPX (Coros GPX) sur votre smartphone.
  2. Ouvrez le fichier sur votre téléphone et sélectionnez l'application COROS App pour l'ouvrir.
  3. Appuyez sur Enregistrer pour l'ajouter à votre bibliothèque d'itinéraires.
  4. Dans la page de détails du tracé, appuyez sur l'icône de montre pour l'envoyer à votre appareil.

À propos de Trail Mapper & POI Merger

Avertissement : Ce projet est bénévole, open-source et indépendant. Il n'est pas affilié à UTMB Group, UTMB World ou toute autre organisation de course.

Trail Mapper & POI Merger est un utilitaire conçu pour les traileurs et passionnés de course de montagne qui naviguent avec des montres GPS Garmin, Suunto ou Coros.

Lors de la préparation d'une course, les organisateurs publient séparément les fichiers GPX et les horaires des ravitaillements. Cet outil résout le problème en recalant automatiquement les points de ravitaillement directement sur la trace GPX.

Caractéristiques principales :

  • Récupération directe : Collez l'URL d'une course (TDS, UTMB, CCC, Templiers, etc.) pour importer automatiquement le tracé GPX officiel et les ravitaillements.
  • Projection automatique : Les points de passage sont mathématiquement recalés sur la coordonnée la plus proche du fichier GPX.
  • Compatibilité Garmin : Raccourcissement automatique des noms (ex. Ravitaillement → RAV), ajout de l'altitude au nom et troncature pour respecter les limites des montres.
  • Formats d'exportation multiples : Téléchargez le fichier GPX compatible Garmin (avec les course points intégrés), le format TCX Garmin, ou GPX Suunto/Coros.
  • Persistance locale : Sauvegarde automatiquement votre parcours actif et vos réglages dans votre navigateur (localStorage).
